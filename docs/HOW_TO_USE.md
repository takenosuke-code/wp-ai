# 使い方ガイド（Loop AI 投稿アシスタント）

AIとチャットしながらブログ記事を作成・公開できるツールです。**「ボタンを押す」でも「チャットに入力する」でも、どちらでも進められます。** 決まった手順どおりに操作できなくても大丈夫なように作ってあります。

## 基本の流れ（全8ステップ）

1. **内容を伝える** — 「◯◯について記事を書きたい」と話しかけます。AIが目的（集客・SEO・情報提供など）や読者を確認します。
2. **画像をアップ**（任意） — 入力欄の画像ボタン、またはプレビュー上の「＋」から写真を追加できます。追加しなくても進めます。
3. **AI要約・確認** — AIが「これでいいですか？」と内容を整理して確認します。「OK」で下書き作成へ。
4. **SEOチェック**（任意） — 競合を実際にWeb検索して、改善点とキーワードを提案します。
5. **プレビュー** — 「公開後の見た目」を確認します。画像もここで差し替え・追加できます。
6. **タイトル設定** — タイトル・URL・カテゴリ・タグ・抜粋を整えます。
7. **スケジュール** — 今すぐ公開するか、日時を指定して予約します。
8. **公開** — 最終確認のうえ「公開する」で公開サイトに反映されます。

## 「ボタンが分からない／押したくない」ときは

- **どのステップでも、チャットに普通に入力すればOKです。** 例：「OKで」「これで進めて」「タイトルはこれでいい」など。
- **「おまかせで進めて」** ボタン（またはそう入力）を使うと、それまでに伝えた内容をもとに、AIが常識的な前提で不足を補い、余計な質問をせずに先へ進めます。SEOの質問などが多いと感じたら、これを押してください。**勝手に事実を捏造したりはしません**（伝えていない具体情報は一般的な範囲でまとめます）。
- 下書きができたあとに **「公開して」「最後まで進めて」** と入力すると、公開の手前（最終確認）まで自動で進みます。**確認なしにいきなり公開はしません。**

## 画像について

- 画像はチャットの画像ボタン、またはプレビューの各セクションの「＋」から追加します。
- 追加した画像はAIが内容を見て、記事の最適な位置に自動で配置します（あとから差し替え・削除も可能）。
- 画像の追加・公開はAPIコストがかかりません。

## 会社・サービスを紹介する記事を書くとき（URLで正確に）

- 特定の会社やサービスを宣伝する記事では、**公式サイトのURLをチャットに貼ってください。** AIがそのページを実際に読み込み、書かれている事実（サービス内容・特徴など）をもとに記事を作ります。
- これにより、**存在しないサービスや実績をAIが勝手に作り出す（ハルシネーション）のを防ぎます。** ページに書かれていないことは、無理に断定せず「[[要確認: ○○]]」として残すか、あなたに確認します。
- URLがない場合は、要点を箇条書きでチャットに伝えてもらえれば、その範囲で記事を作ります。

## 「もう戻れないかも？」と不安なとき

- **公開する前なら、いつでもやり直せます。** 各画面の「キャンセル」「戻る」で前のステップに戻れます。
- 公開の最終確認画面で気が変わったら、「キャンセルして修正」を押してチャットで修正をお願いできます（**その時点ではまだ公開されていません**）。
- 画面の表示が途中で崩れたりエラーが出ても、**次のメッセージを送れば自動で復旧します**（会話の状態は自動で修復されます）。同じ会話を続けて大丈夫です。

## こんなときは

- **「この会話はまだ処理中です」と出た** — 前の返信の生成が終わるまで少し待ってから送ってください（同じ会話を2つのタブで同時に操作すると出ることがあります）。
- **赤いエラーが一度出た** — もう一度メッセージを送れば復旧します。繰り返し出る場合は新しいチャットを始めてください。
- **公開したのに反映されない** — 予約公開にしていないか、右上「公開済み」で確認してください。予約は指定時刻まで公開サイトに表示されません。
